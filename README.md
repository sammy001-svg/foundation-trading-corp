# FOUNDATION TRADING COMPANY | Global Commodity Trading & Logistics

FOUNDATION TRADING COMPANY is a premier global commodity trading house, facilitating the flow of essential resources across continents with integrity, precision, and operational excellence. This platform serves as a production-ready corporate portal and intelligence hub for global trade operations.

## 🚀 Key Features

- **Dynamic Corporate Experience**: High-contrast, brand-centric UI with smooth animations (`framer-motion`).
- **Interactive Trade Network**: Custom SVG-based global trade map visualizing connectivity centered on Nairobi.
- **Market Intelligence**: Dedicated insights portal with data-driven research and strategic analysis.
- **Operational Precision**: Functional contact systems backed by Next.js API routes.
- **Performance Optimized**: Optimized asset delivery using `next/image` for high-performance global access.
- **SEO Ready**: Dynamic sitemaps, robots.txt, and complete OpenGraph/Twitter social metadata.

## 🛠️ Technology Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Deployment Ready**: Standardized production build workflow.

## 📁 Project Structure

```text
src/
├── app/            # Next.js App Router (Pages, API, Layouts)
├── components/     # Reusable UI components (Home, Layout, Shared)
├── data/           # Centralized JSON content management
└── lib/            # Utilities and core logic
public/             # High-resolution optimized assets
```

## 🏗️ Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
